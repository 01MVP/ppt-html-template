# HTML PPT 模板

一个纯HTML + CSS + JavaScript实现的演示文稿模板，无需安装任何依赖，开箱即用，支持AI生成内容。

## ✨ 特性

### 🚀 开箱即用
- **零依赖安装** - 无需Node.js、Vue等环境，下载解压即可使用
- **纯前端实现** - 使用标准的HTML、CSS、JavaScript技术
- **新手友好** - 简单的文件结构，详细的注释说明

### 📱 响应式设计
- **跨设备适配** - 桌面端完整PPT体验，移动端优化布局
- **触摸支持** - 支持手势滑动切换幻灯片
- **多分辨率** - 适配各种屏幕尺寸和分辨率

### 🎨 主题系统
- **Apple风格** - 简洁、大量留白、优雅动画、现代感强
- **极简风格** - 黑白配色、几何图形、大字体、高对比度
- **新野兽派** - 大胆色彩、粗体字、不规则布局、强视觉冲击

### ⌨️ 强大功能
- **完整键盘导航** - 支持方向键、空格、F11全屏等快捷键
- **演讲者模式** - 按S键开启，显示备注和当前进度
- **全屏演示** - F11进入全屏，ESC退出
- **幻灯片管理** - 侧边栏导航，进度指示器

### 🤖 AI 友好
- **提示词模板** - 内置多种风格的AI生成提示词
- **结构化生成** - 清晰的文件组织，便于AI理解和生成
- **风格指导** - 为不同主题提供具体的设计指引

## 🛠️ 自定义修改

### 修改配置
编辑 `config.js` 文件：
```javascript
const PPTConfig = {
    // 修改标题
    title: "你的演示标题",
    
    // 修改作者
    author: "你的名字",
    
    // 修改幻灯片信息
    slides: [
        {
            id: "your-slide",
            title: "你的幻灯片标题",
            layout: "content",
            notes: "演讲者备注"
        }
    ],
    
    // 修改设置
    settings: {
        autoplay: false,        // 是否自动播放
        transition: "slide",    // 过渡效果
        showControls: true,     // 显示控制按钮
        keyboard: true          // 启用键盘导航
    }
};
```

### 添加新幻灯片
1. 在 `index.html` 中添加新的幻灯片 div
2. 在 `config.js` 中添加对应的配置
3. 在侧边栏导航中添加导航项

### 自定义样式
- 修改 `assets/css/main.css` 调整基础样式
- 修改主题文件 `assets/css/themes/` 调整主题样式
- 添加自定义CSS变量

### 扩展功能
- 在 `assets/js/main.js` 中添加新功能
- 使用 `keyboardController.addShortcut()` 添加快捷键
- 监听PPT状态变化：`PPTState`

## 🔧 故障排除

### 常见问题

**Q: 幻灯片切换没有动画效果？**
A: 检查浏览器是否支持CSS3动画，或在设置中关闭了动画效果。

**Q: 键盘快捷键不工作？**
A: 确保页面获得了焦点，点击页面任意位置后再试。

**Q: 移动端显示异常？**
A: 检查viewport设置，确保使用了响应式样式。

**Q: 主题切换失败？**
A: 检查主题CSS文件路径是否正确，确保文件存在。
