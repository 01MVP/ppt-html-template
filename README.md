# 📊 HTML PPT 模板

一个零依赖的纯HTML PPT演示模板，支持16:9比例显示和丰富的功能。

## ✨ 特性

- 🚀 **零依赖**：直接打开`index.html`即可使用，无需本地服务器
- 📱 **响应式设计**：支持桌面、平板和手机设备
- 🎯 **16:9比例**：专业的PPT显示比例，适合各种演示场景
- 🎨 **样式解耦**：编辑器样式与内容样式完全分离
- ⚡ **功能丰富**：演示计时器、幻灯片概览、布局模板等
- 📝 **易于配置**：所有内容在`config.js`中配置，无需外部文件

## 🚀 快速开始

### 1. 下载并打开

1. 下载或克隆项目
2. 双击打开`index.html`文件
3. 立即开始使用，无需任何设置

### 2. 编辑内容

1. 打开`config.js`文件
2. 在`slideContent.sets.custom.slides`数组中添加你的幻灯片
3. 修改`slideContent.current`为`"custom"`切换到自定义内容

### 3. 幻灯片配置示例

```javascript
// 在 config.js 中添加幻灯片
slideContent: {
    current: "custom", // 使用自定义内容
    sets: {
        custom: {
            title: "我的演示",
            description: "我的自定义演示文稿",
            slides: [
                {
                    id: "welcome",
                    title: "欢迎",
                    content: `
                        <div class="slide-content cover-layout">
                            <div class="cover-main">
                                <h1 class="cover-title">我的演示标题</h1>
                                <p class="cover-subtitle">演示副标题</p>
                            </div>
                        </div>
                    `,
                    notes: "这是演讲者备注"
                },
                // 更多幻灯片...
            ]
        }
    }
}
```

## 🎛️ 功能面板

按`Ctrl + /`打开功能面板，包含以下功能：

### 📊 演示工具
- **演示计时器** (`Alt + T`)：精确的演示时间管理
- **演讲者模式** (`S`)：查看演讲者备注
- **全屏演示** (`F11`)：全屏显示模式
- **黑屏模式** (`B`)：临时黑屏

### 🔧 编辑工具
- **幻灯片概览** (`Alt + O`)：缩略图导航和拖拽排序
- **布局模板** (`Alt + L`)：10种专业布局模板
- **代码高亮**：自动语法高亮
- **搜索功能** (`Ctrl + F`)：全文搜索

## 🎯 16:9 比例设计

本模板专为16:9比例设计，确保在各种设备上都能保持专业的演示效果：

- **桌面端**：自动适配屏幕尺寸，保持16:9比例
- **移动端**：响应式布局，优化触摸操作
- **投影仪**：完美适配标准投影设备

## 🎨 样式自定义

编辑器样式与内容样式完全分离：

- **编辑器样式**：`assets/css/editor.css`（控制功能面板、工具栏等）
- **内容样式**：`assets/css/main.css`（控制幻灯片内容显示）
- **主题样式**：`assets/css/themes/`（不同的配色方案）

## 🆘 常见问题

**Q: 为什么不能正常显示内容？**
A: 确保直接打开`index.html`文件，检查`config.js`中的配置是否正确。

**Q: 如何添加自定义幻灯片？**
A: 编辑`config.js`文件，在`slideContent.sets.custom.slides`数组中添加幻灯片对象。

**Q: 可以离线使用吗？**
A: 是的，除了字体和图标CDN，所有功能都可以离线使用。
